# PXScript

PXScript is an interpreted language.

Now there is only a file, named interpreter, wrote in Ruby. 

## Installation

To install PXScript in your computer, you need to have Ruby installed.

After you cloned the Repository, use command 

> sudo make install

to add the interpreter to the system PATH.

Note: I developed the repo on Ubuntu WSL, so it may not work on any distro. Feel free to create a commit!

## How to use

To try it, create a file, put in some PXScript code, and then run the command

> pxscript (file name)

and it will work (i hope)
